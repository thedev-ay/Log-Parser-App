# Log Parser App
A command line node.js application, which parses the input log file

## Setup
```bash
$ git clone https://github.com/thedev-ay/Log-Parser-App.git
$ cd Log-Parser-App
$ npm install
$ npm run build
```

## Testing
```bash
$ npm run start or
$ node lib/index.js --input \"logs/app.log\" --output \"errors.json\"
```
